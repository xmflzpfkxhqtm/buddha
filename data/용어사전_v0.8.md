## 사용 원칙

1. **고유명사(인명·지명)**: 한국 불교 전통 음역 원칙. 국제적으로 더 잘 알려진 산스크리트 음역이 있는 경우 그것을 채택하고 전통 음역은 불채택 표기란에 기재.
2. **불교 개념**: 한문 원어 표기 유지. 측량·수치 단위 포함.
3. **동일 개념 중복 표기 금지**: 여러 이름으로 불리는 개념은 가장 쉽고 보편적인 하나로 통일. 나머지는 불채택 표기란에 기재.
4. **출처 경전 표기**: 본문 안 인용 시 『』 사용.

## 4. 문체 원칙

### 기본
- **게송(偈頌)**: 본문과 구분하여 들여쓰기 처리. 운율 살리되 의미 최우선.
- **주석·별기**: 기존 번역의 괄호 주석은 본문 흐름을 끊지 않도록 자연스럽게 흡수하거나 생략.
- **인용 경전 출처 표기**: 문단 말미에 줄을 바꿔 `— 출처` 형식으로 통일.
- **숫자**: 천문학적 수치(유순, 겁 수)는 아라비아 숫자 사용 (예: 4만 2천 유순).

### 현대화 원칙 (v0.2 추가)

**원칙 1. 낯선 역사·문화 비유는 제거**
한국 독자에게 배경지식이 없으면 이해하기 어려운 역사적 인물이나 고사 비유는 생략하거나 의미만 남긴다.
- 예) "유향(劉向)이 서고의 책을 교정하듯" → 비유 전체 삭제, 결과 의미만 단문으로 서술

**원칙 2. 불교 고유어가 아닌 한자 합성어는 현대 한국어로**
불교 고유 개념어(삼장, 선열, 풍륜 등)는 원어 유지. 그 외 일상에서 쓰이지 않는 한자 합성어는 뜻이 분명한 현대어로 풀어 쓴다.
- 예) 법속(法俗) → "승려와 재가신자", 원림(園林) → "동산", 천의(天衣) → "옷자락"

**원칙 3. 긴 문장은 핵심만 담은 단문으로**
여러 절을 이어 붙인 문장은 핵심만 남겨 짧은 단문으로 나눈다. 내용 일부 생략도 허용.
- 예) "경전을 두루 초록하고 심오한 경문의 증거를 드러내되 신묘한 종지를 집약하고 말은 간략하게 뜻은 분명히 하도록 하셨다" → "경전을 두루 초록하되 핵심만을 간결하게 정리하도록 한 것이다"

**원칙 4. 간접·문학적 표현보다 직접 표현 우선**
의미를 에두르거나 문학적으로 꾸민 표현보다 뜻이 즉각 전달되는 직접적 표현을 쓴다.
- 예) "아직 표시를 드러내지 못하고 있었다" → "아직 정리되지 않은 채였다", "천의를 드리워" → "옷자락을 내려"
