불설상응상가경

서진(西晉) 승려 법촉(法炬) 역

부처님께서 사위국(舍衛國)에서 여러 비구들을 청하셨다. 비구들이 즉시 도착하니, 부처님께서 비구들에게 말씀하셨다. 비구들이 대답하였다. “예, 부처님께서 들으라 하시니 듣겠습니다.”

부처님께서 곧 말씀하셨다. “듣지 않는 자들은 듣지 않는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 많이 듣는 자들은 많이 듣는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

탐욕하는 자들은 탐욕하는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 탐욕하지 않는 자들은 탐욕하지 않는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

진노하는 자들은 진노하는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 진노하지 않는 자들은 진노하지 않는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

어리석은 자들은 어리석은 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 지혜로운 자들은 지혜로운 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

보시하는 자들은 보시하는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 인색한 자들은 인색한 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

욕망이 적은 자들은 욕망이 적은 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 욕망이 많은 자들은 욕망이 많은 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

계율을 지키지 않는 자들은 계율을 지키지 않는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 계율을 지키는 자들은 계율을 지키는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

나누기 어려운 자들은 나누기 어려운 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 나누기 쉬운 자들은 나누기 쉬운 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

만족할 줄 모르는 자들은 만족할 줄 모르는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 만족할 줄 아는 자들은 만족할 줄 아는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.

스스로를 지키지 않는 자들은 스스로를 지키지 않는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다. 스스로를 지키는 자들은 스스로를 지키는 자들끼리 서로 닮고, 모여서, 서로 어울리며, 서로 맞는다.”

부처님께서 말씀하셨다. “비구들이여, 이와 같이 지혜로운 자는 이 인연을 분별하여 행할 수 있는 것은 행하고, 행할 수 없는 것은 행하지 말아야 한다.”

부처님께서 이와 같이 말씀하셨다. 비구들이 받아 행하고 마음에 새겼다.
