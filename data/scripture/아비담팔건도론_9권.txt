아비담팔견도론 권 제9

가섭연자(迦旃延子) 저술

진(秦)·계빈(罽賓) 삼장 승가제파(僧伽提婆)와 축불념(竺佛念) 공역

지견도(智犍度) 제3

오발거(五跋渠) 송(頌)에서 이르기를:

여덟 가지 지혜는 다섯 처소가 있고,

타인의 마음과 지혜를 알며,

만약 수행지혜를 닦으면,

상응하는 것은 가장 뒤에 있다.

아비담팔도발거(阿毘曇八道跋渠) 제1

여든 가지 지혜는,

항상 앞에 머물러 있으며,

모든 상응하는 깨달음은,

세속견(世俗見) 무루(無漏)이다.

또한 세존께서 말씀하시기를: 여덟 가지 성취한 학적(學迹)과 열 가지 무루(漏) 소멸한 아라한이 있다. 여덟 가지 학적은 과거, 미래, 현재 각각 몇 가지 성취하는가? 열 가지 무루 소멸 아라한은 과거, 미래, 현재 각각 몇 가지 성취하는가?

어떻게 견(見)이라 하는가? 어떻게 지(智)라 하는가? 어떻게 혜(慧)라 하는가? 만약 견이라면, 그것이 지인가? 만약 지라면, 그것이 견인가? 만약 견이라면, 그것이 혜인가? 만약 혜라면, 그것이 견인가? 만약 지라면, 그것이 혜인가? 만약 혜라면, 그것이 지인가? 견이 지를 포섭하는가, 지가 견을 포섭하는가? 견이 혜를 포섭하는가, 혜가 견을 포섭하는가? 지가 혜를 포섭하는가, 혜가 지를 포섭하는가? 만약 견을 성취한다면, 그것이 지인가? 만약 지를 성취한다면, 그것이 견인가?

만약 견을 성취한다면, 그것이 혜인가? 만약 혜를 성취한다면, 그것이 견인가? 만약 지를 성취한다면, 그것이 혜인가? 만약 혜를 성취한다면, 그것이 지인가? 만약 견이 이미 멸하여 남김이 없다면, 그것이 지인가? 만약 지가 이미 멸하여 남김이 없다면, 그것이 견인가? 만약 견이 이미 멸하여 남김이 없다면, 그것이 혜인가? 만약 혜가 이미 멸하여 남김이 없다면, 그것이 견인가? 만약 지가 이미 멸하여 남김이 없다면, 그것이 혜인가? 만약 혜가 이미 멸하여 남김이 없다면, 그것이 지인가?

질문, 정의, 포섭, 성취, 멸. 소위 등견(等見)은 선택법(擇法) 각의(覺意)인가? 만약 선택법 각의라면, 그것이 등견인가? 소위 등지는 선택법 각의인가? 만약 선택법 각의라면, 그것이 등지인가? 염각의가 현재 앞에 있을 때, 몇 각의, 몇 도종(道種)이 현재 앞에 있는가? 정진, 희, 의(猗), 정, 호각의, 등견, 등지, 등어, 등업신, 등명, 등방편, 등념, 등정이 현재 앞에 있을 때, 몇 각의, 몇 도종이 현재 앞에 있는가?

제법(諸法)이 염각의와 상응한다면, 그것이 선택법 각의인가? 만약 제법이 선택법 각의와 상응한다면, 그것이 염각의인가? 제법이 염각의와 상응하고, 정진, 희, 의, 정, 호각의, 등견, 등지, 등방편, 등념, 등정과 상응한다면, 그것이 염각의인가? 만약 제법이 등정과 상응한다면, 그것이 염각의인가? 제법이 등념과 상응한다면, 그것이 등정인가? 만약 제법이 등정과 상응한다면, 그것이 등념인가? 세속 등견은 어떻게 되는가?

세속 등지는 어떻게 되는가? 만약 세속 등견이라면, 그것이 세속 등지인가? 만약 세속 등지라면, 그것이 세속 등견인가? 세속 등견이 세속 등지를 포섭하는가? 만약 세속 등지라면, 그것이 세속 등견을 포섭하는가? 만약 성취한 세속 등견이라면, 그것이 세속 등지인가? 만약 성취한 세속 등지라면, 그것이 세속 등견인가? 만약 세속 등견이 이미 멸하여 남김이 없다면, 그것이 세속 등지인가? 만약 세속 등지가 이미 멸하여 남김이 없다면, 그것이 세속 등견인가?

질문, 정의, 포섭, 성취, 멸. 어떻게 무루 등견인가? 어떻게 무루 등지인가? 만약 무루 등견이라면, 그것이 무루 등지인가? 만약 무루 등지라면, 그것이 무루 등견인가? 무루 등견이 무루 등지를 포섭하는가? 만약 무루 등지라면, 그것이 무루 등견을 포섭하는가? 만약 성취한 무루 등견이라면, 그것이 무루 등지인가? 만약 성취한 무루 등지라면, 그것이 무루 등견인가? 질문, 정의, 포섭, 성취.

이 장의 뜻을 원만히 설한다. 또 세존께서 말씀하시기를: 여덟 가지 성취한 학적과 열 가지 무루 소멸한 아라한이 있다. 여덟 가지 학적은 과거, 미래, 현재 각각 몇 가지 성취하는가?

답하되: 만약 유각유관 삼매학(有覺有觀三昧學) 초견 현재 앞이라면, 그것은 과거가 아니며, 여덟 가지 미래, 여덟 가지 현재이며, 멸한 후에도 잃지 않는다. 만약 유각유관 삼매학 견 현재 앞이라면, 그것은 여덟 가지 과거 성취, 여덟 가지 미래, 여덟 가지 현재이며, 멸한 후에도 잃지 않는다. 만약 무각무관 삼매학 견 현재 앞이라면, 그것은 여덟 가지 과거, 여덟 가지 미래, 일곱 가지 현재이며, 등지가 없다.

멸한 후에도 잃지 않는다. 만약 무색정 학 견 현재 앞이라면, 그것은 여덟 가지 과거, 여덟 가지 미래, 네 가지 현재이며, 등지, 등어, 등업, 등명 무색의 뜻을 제외한다. 멸한 후에도 잃지 않는다. 만약 열반삼매에 들어가 세속심 현재 앞이라면, 그것은 여덟 가지 과거, 여덟 가지 미래, 현재가 없다. 유각유관이 끝난다.

만약 무각무관 삼매학 초견 현재 앞이라면, 그것은 과거가 없고, 미래 여덟, 현재 일곱이며, 멸한 후에도 잃지 않는다. 다시 무각무관 삼매학 견 현재 앞이라면, 그것은 과거 일곱, 미래 여덟, 현재 일곱이며, 멸한 후에도 잃지 않는다. 만약 무색정 학 견 현재 앞이라면, 그것은 과거 일곱, 미래 여덟, 현재 네 가지이며, 멸한 후에도 잃지 않는다. 만약 열반정에 들어가 세속심 현재 앞이라면, 그것은 과거 일곱, 미래 여덟, 현재가 없으며, 멸한 후에도 잃지 않는다. 다시 유각유관 삼매학 견 현재 앞이라면, 그것은 과거 일곱, 미래 여덟, 현재 여덟이다. 무각무관이 끝난다.

만약 무색정 학 초견 현재 앞이라면, 그것은 과거가 없고, 미래 여덟, 현재 네 가지이며, 멸한 후에도 잃지 않는다. 다시 무색정 학 견 현재 앞이라면, 그것은 과거 네 가지, 미래 여덟, 현재 네 가지이며, 멸한 후에도 잃지 않는다. 다시 열반정에 들어가 세속심 현재 앞이라면, 그것은 과거 네 가지, 미래 여덟, 현재가 없으며, 멸한 후에도 잃지 않는다.

만약 멸한 후에도 잃지 않는다면, 다시 유각유관 삼매학 견 현재 앞이라면, 그것은 과거 네 가지, 미래 여덟, 현재 여덟이며, 멸한 후에도 잃지 않는다. 다시 무각무관 삼매학 견 현재 앞이라면, 그것은 과거 네 가지, 미래 여덟, 현재 일곱이다. 학적이 끝난다.

열 가지 무루 소멸 아라한은 과거, 미래, 현재 각각 몇 가지 성취하는가? 답하되: 만약 유각유관 삼매 무학 초지 현재 앞이라면, 그것은 과거가 없고, 미래 열 가지, 현재 아홉이며, 등견이 없다. 멸한 후에도 잃지 않는다. 다시 유각유관 삼매 무학 지 현재 앞이라면, 그것은 과거 아홉, 미래 열 가지, 현재 아홉이며, 멸한 후에도 잃지 않는다. 만약 무각무관 삼매 무학 지 현재 앞이라면, 그것은 과거 아홉, 미래 열 가지, 현재 여덟이다. 등지, 등견이 없다.

멸한 후에도 잃지 않는다. 만약 무색정 무학 지 현재 앞이라면, 그것은 과거 아홉, 미래 열 가지, 현재 다섯이며, 초상 무계(無戒)이다. 멸한 후에도 잃지 않는다. 만약 열반정에 들어가 세속심 현재 앞이라면, 그것은 과거 아홉, 미래 열 가지, 현재가 없다. 지가 끝난다.

멸한 후에도 잃지 않는다. 만약 유각유관 삼매 무학 초견 현재 앞이라면, 그것은 과거 아홉, 미래 열 가지, 현재 아홉이며, 멸한 후에도 잃지 않는다. 다시 유각유관 삼매 무학 지견 현재 앞이라면, 그것은 과거 미래 열 가지, 현재 아홉이다.

멸한 후에도 잃지 않는다. 다시 무각무관 삼매 무학 지견 현재 앞이라면, 그것은 과거 미래 열 가지, 현재 여덟이며, 멸한 후에도 잃지 않는다. 다시 무색정 무학 지견 현재 앞이라면, 그것은 과거 미래 열 가지, 현재 다섯이며, 멸한 후에도 잃지 않는다. 다시 열반정에 들어가 세속심 현재 앞이라면, 그것은 과거 미래 열 가지, 현재가 없다. 유각관이 끝난다.

만약 무각무관 삼매 무학 초견 현재 앞이라면, 그것은 과거가 없고, 미래 열 가지, 현재 여덟이며, 멸한 후에도 잃지 않는다. 다시 무각무관 삼매 무학 견 현재 앞이라면, 그것은 과거 여덟, 미래 열 가지, 현재 여덟이며, 멸한 후에도 잃지 않는다. 다시 무색정 무학 견 현재 앞이라면, 그것은 과거 여덟, 미래 열 가지, 현재 다섯이다.

멸한 후에도 잃지 않는다. 다시 열반정에 들어가 세속심 현재 앞이라면, 그것은 과거 여덟, 미래 열 가지, 현재가 없으며, 멸한 후에도 잃지 않는다. 다시 유각유관 삼매 무학 견 현재 앞이라면, 그것은 과거 여덟, 미래 열 가지, 현재 아홉이다. 견이 끝난다.
